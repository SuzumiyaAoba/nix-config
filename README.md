# nix-config

My Nix configuration.

## Install

```console
$ curl --proto '=https' --tlsv1.2 -sSf -L https://raw.githubusercontent.com/SuzumiyaAoba/nix-config/master/init.sh | sh -s
```

## Commands

### for Private

```console
$ makers private:apply
```

### for Work

```console
$ makers private:apply
```

### format

```console
$ nix --extra-experimental-features nix-command --extra-experimental-features flakes fmt
```
